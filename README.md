# 🚀 Azure Terraform Module

## 📌 Description
This project contains Terraform modules to provision Azure infrastructure including:

- Resource Group
- Virtual Network (VNet)
- Storage Account

It helps in automating infrastructure deployment using Infrastructure as Code (IaC).

---

## 🛠️ Technologies Used

- Terraform
- Microsoft Azure
- HCL (HashiCorp Configuration Language)
